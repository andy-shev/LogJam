Dependencies

LogJam depends on GTK >= 2.4.

It can use GTKSpell, SQLite, libgtkhtml3, libcurl, and XMMS if you have them, but they're not required.

Source

LogJam can be downloaded here:

  • logjam-4.5.3.tar.bz2 [749KB] 19 Jan 2006

Binaries

  • Debian: apt-get install logjam
  • RedHat: RPMs, including dependencies, are collected at Tom's page.
  • FreeBSD, OpenBSD: in the ports tree.
