Development

LogJam has been mostly abandoned for years, but the source is now on github in the hopes that someone will fork it.

Keeping Up

We don't have a mailing list. Instead, there are two LiveJournals to follow:

  • logjam_dev is where we discuss development issues (bugs, design, etc.), and
  • logjam_cvs (though we don't use CVS anymore) logs commits to the repository.
